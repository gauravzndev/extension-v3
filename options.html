Configure File Naming

Decide exactly what your downloaded files and folders are called. Drag the coloured pills below into a formula, rearrange them, and double-click any pill to remove it. The live preview updates as you build, so you can see the final filename before saving.

Opens your browser's built-in prompt every time you click Download, pre-filled with the post's title so you can keep it as-is by pressing Enter, or type a new name to override the title for that one gallery only. The rest of your formula (subreddit, index, date, etc.) is still applied around it. Leave this off to download silently using the post's real title.
Press Alt + D on any Reddit gallery page to start the download without reaching for the mouse. The shortcut stays out of the way while you're typing in comment boxes or search fields. If your browser uses Alt + D to focus the address bar, turn this off to avoid the conflict.

Pick how a gallery lands on your disk. Folder creates a sub-folder per gallery and drops the images inside. ZIP bundles the whole gallery into a single archive. Individual saves the images straight into your base downloads folder with no wrapper. Each mode has its own naming formula below — switch tabs to configure them.
