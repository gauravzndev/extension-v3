Tips & recipes

Seven small habits that pay off forever. Each one is a pattern I keep coming back to when I'm bulk-saving Reddit galleries — copy any of them straight into your own setup.

tip one.

One folder per subreddit

Every gallery from r/photography lands in the same photography/ folder regardless of which post it came from. Build the folder formula with just one pill — Subreddit — and you're done.

set this up.

  1. Open the settings page → Folder Mode
  2. In Folder Naming Formula, drop only Subreddit
  3. In Image Filename Formula, use Title + Index so the individual images stay distinguishable
tip two.

Never overwrite a file again

Two posts with the same title (or no title at all) downloaded into the same folder will normally collide. Add a Unique ID pill at the end of the image filename and every download is guaranteed unique.

set this up.

  1. In Image Filename Formula, drop Unique ID at the end
  2. Pick the format you like under Format tweaks → Unique ID Format (hex, numeric, timestamp...)
tip three.

Group by upload date

Want a chronological archive — everything uploaded on a given day in one folder? Put Upload Date in your folder formula. Use Year, Month, Day so folders sort cleanly in File Explorer.

set this up.

  1. Open Folder Mode
  2. Drop Upload Date as the first pill in Folder Naming Formula
  3. Under Format tweaks → Date Format, choose Year, Month, Day
tip four.

One file to share the whole gallery

Switch to ZIP Mode and a 12-image post becomes a single .zip you can drag into Discord, attach to email, or upload anywhere. Recipients unzip once and they have the entire post intact.

set this up.

  1. Click the extension's puzzle icon → Download Mode → Zip Posts
  2. (Optional) Open the settings page → ZIP Mode → tweak the archive name formula
tip five.

Move your setup to another machine

Spent an hour perfecting your naming formula? Don't redo it on your laptop. Export your config to a JSON file, copy it across, and import it on the other browser — all your formulas, themes, and toggles land identical.

set this up.

  1. Settings page → Backup & reset → Export Settings
  2. Move the downloaded .json to the other computer
  3. Settings page on that machine → Import Settings → pick the file
tip six.

Download without touching the mouse

While you're scrolling Reddit, hit Alt + Shift + D on any gallery post to start the download. The combo is deliberately uncommon — most browsers don't bind it, so it sidesteps the address-bar shortcut on plain Alt + D. Stays quiet while you're typing in comment boxes or the search bar.

set this up.

  1. Settings page → top toggle "Enable Alt + Shift + D keyboard shortcut" is on by default
  2. If you find any conflict in your particular setup, the toggle turns the shortcut off cleanly
tip seven.

Edit the custom-title prompt faster

When the custom title prompt opens with the post's title pre-filled, you don't have to nudge the cursor one character at a time. Hold Ctrl with ← or → to jump a whole word at a time — saves a lot of arrow-key tapping on long titles.

when this helps.

  1. Settings page → toggle "Prompt for Custom Title before downloading" on
  2. Click Download Gallery — the prompt opens with the existing title pre-filled
  3. Use Ctrl + ← / → to land your cursor between words instantly, then edit just the part you want
